      8. 연산자
         1. 할당 연산자
         -특징: 오른쪽 피연산자의 값을 왼쪽 변수에 할당
         -종류: =, 복합 할당 연산자 (+=,-=,*=,/=,%=)

         2. 산술 연산자
         -특징: 증가, 감소 단항 부정(-), 숫자화(+)
         -종류: +,-,*,/,%(나머지),**(제곱근),++,--

         3. 단항 부정 연산자
         -특징: 부호가 바뀜

         3-2. 숫자화 연산자
         -특징: 숫자가 아닌 문자 boolean 값을 숫자로 변환하는 시도
         
         4. 논리 연산자
         -특징: 조건을 여러 가지 만족 시키는지 알아보는 연ㅈ산자
         -종류: &&(둘다 true일 경우 true), || 둘중 하나

         5. 문자열 관련 연산자
         -특징: 합치기
         -종류 : +

         6. 조건(삼항 연산자)
         -특징 : 조건 ? true일 때 할당할 값 :false 일 때 할당할 값
                중첩해서 사용이 가능하다
